Conclusión

Eduardo Salazar Treviño 1847972

Esta práctica me pareció realmente interesante, es la segunda o tercera vez que trabajo con el elemento Canvas en mi vida y la primera vez que lo hago de forma dinámica recibiendo input del usuario.
Creo que tomando en cuenta que es una API nativa de HTML5 es una muy buena herramienta que sirve para poder satisfacer las necesidades de diseños básicos que tenga un programador web sin recurrir a software externo, e incluso, si se llega a usar con Javascript de una manera extensa se pueden lograr dibujos mucho más complejos e interactivos.
Por otra parte, también es necesario tener buenos conocimientos de conceptos como el DOM para interactuar con este en el script que controle el Canvas, no basta sólo con saber de HTML y CSS. Para mí, esta fue una práctica bastante enriquecedora y que me sirvió para reforzar mis conocimientos de Javascript
