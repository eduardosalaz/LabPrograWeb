Conclusión

Eduardo Salazar Treviño 1847972

Esta práctica me pareció la más interesante de todas, nunca antes en mi vida había trabajado con la API de Drag and Drop de HTML5 y para ser la primera vez que lo hago, se me hizo sencilla, si bien existen mejores maneras de trabajar con elementos arrastrables , como lo pueden ser jQuery, considero que si el navegador nos da la posibilidad de trabajar con herramientas nativas, hay que hacerlo.
Creo que tomando en cuenta que es una API nativa de HTML5 es una muy buena herramienta que sirve para poder satisfacer las necesidades de arrastrar y soltar elementos que tenga un programador web sin recurrir a software externo, e incluso, si se llega a usar con Javascript de una manera extensa y bien escrita se pueden escribir programas más complejos usando sólo las herramientas que ya incluyen los navegadores que soportan HTML5.
Para mí, esta fue una práctica bastante enriquecedora y que me sirvió para reforzar mis conocimientos de Javascript
