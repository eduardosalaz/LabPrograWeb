Práctica 5 de Laboratorio de Programación Web

Explicación del código de la Práctica 5

Para esta práctica se usaron las siguientes tags

Explicación de cada una

  • <h1> : Es la tag que marca los encabezados dentro de nuestro contenido, mientras más alto sea el número menor será el encabezado.
  • <div>: Es usada para agrupar elementos de html para poder manipularlos con CSS o estructurar el contenido de la página. Debe de tener un nombre de div o clase de CSS.
  • <nav>: Es usada para marcar la sección de navegación de la página donde se ubican los enlaces a otras páginas.
  • <audio>: Es usada para poder incluir archivos de audio de manera nativa en la página gracias a las nuevas tecnologías de HTML5. Se especifica la ubicación del archivo, si tiene controles o no, el tipo de archivo, si entra en loop, si está muteado por defecto o si hace autoplay.
  • <video>: Es usada para incluir archivos de video en la página sin necesidad de plugins externos. Se especifica la ubicación del archivo, si tiene controles o no, la altura, el ancho, si entra en loop, si está muteado por defecto o si hace autoplay.
  • <i>: Es usada para hacer itálicas las palabras contenidas dentro de la tag.
  • <strong>: Es usada para remarcar en negritas las palabras contenidas dentro de la tag.
  • <figure>: Es usada para marcar figuras como imágenes o videos dentro de la página para poder añadirles leyendas. Se especifica la figura a añadir, su ubicación y su leyenda con <figcaption>
  • <main>: Es usada para marcar el área principal de la página donde se ubicará el contenido relevante.
  • <footer>: Es usada para marcar el área del pie de página.
